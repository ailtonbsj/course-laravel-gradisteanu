# Course Introduction to Laravel 4 by Gradisteanu

## Requirements

- php7
- mcrypt
- composer

## Commands to create

```bash
composer create-project laravel/laravel myfirstapp 4.2 --prefer-dist

cd ./myfirstapp

php artisan serve

# edit ./app/config/app.php to enable debug and disable error reporting
```

## Commands to use

```bash
cd ./myfirstapp

composer install
```
